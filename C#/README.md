# Rider(Visual Studio Keymap) 단축키
  
### 유틸리티
- Ctrl + T : 모든 파일 보기
- Ctrl + D : 라인 복제
- Ctrl + R,R : 이름 변경
- shift + Alt : 여러 줄 선택
- Ctrl + Alt + arrow : 라인 이동

## # 디버그 관련
- F5 : 실행
- F10 : 디버그로 실행
- F12 : 정의부분 찾기
- Ctrl + '-' : 이전 커서 위치로 이동

### 코드 스타일
Alt + Enter : 자동 코드 정리
File - Setting - Editor - Code Style - C# - Braces Layout
